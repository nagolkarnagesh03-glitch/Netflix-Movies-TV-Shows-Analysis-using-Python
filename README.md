# 🎬 Netflix Movies & TV Shows Analysis using Python

## 📌 Project Overview

This project analyzes the Netflix Movies & TV Shows dataset using Python. The main objective is to perform data cleaning, exploratory data analysis (EDA), and generate meaningful business insights from the dataset.

The project demonstrates practical data analysis skills using Python libraries such as Pandas, NumPy, and Matplotlib.

---

## 🎯 Objectives

* Clean and preprocess the Netflix dataset.
* Handle missing values and duplicate records.
* Perform Exploratory Data Analysis (EDA).
* Answer business-related questions using Python.
* Visualize key insights with charts.

---

## 🛠️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib

---

## 📂 Dataset

* Netflix Movies & TV Shows Dataset
* Source: Kaggle

---

## 🧹 Data Cleaning Process

* Imported the dataset using Pandas.
* Checked dataset structure and data types.
* Identified and handled missing values.
* Removed duplicate records.
* Converted the `date_added` column to datetime format.
* Created new columns such as `year_added`, `month_added`, and `day_added`.
* Split the `duration` column into `duration_value` and `duration_type`.
* Cleaned text columns by removing extra spaces.

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

* Movies vs TV Shows distribution
* Content rating distribution
* Top content-producing countries
* Top directors
* Most common genres
* Content added by year
* Release year analysis
* Movies produced by different countries
* Titles released before the year 2000

---

## ❓ Business Questions Answered

1. How many Movies and TV Shows are available on Netflix?
2. Which content rating is most common?
3. Which countries produce the most Netflix content?
4. Who are the top directors on Netflix?
5. Which genres appear most frequently?
6. Which year had the highest number of titles added to Netflix?
7. Which release years have the most titles?
8. Which countries produce the most Movies?
9. How many titles were released before 2000?

---

## 📈 Key Insights

* Netflix's catalog contains more Movies than TV Shows.
* A few countries contribute a significant share of Netflix content.
* Certain genres appear much more frequently than others.
* Netflix experienced rapid content growth in specific years.
* Most titles belong to a limited set of content ratings.
* The number of titles released before 2000 is relatively small compared to recent releases.
